Start Here

This website is the user manual for our ECSE 2920 Electronics Tool Bench. It is written for a first-time user and explains how to safely power the system, move through the menu, and use each measurement and signal-generation mode.

Before You Begin

  • Make sure the Raspberry Pi and the 24 V supply are connected correctly.
  • Make sure all external test leads are connected to the correct labeled terminals.
  • Do not move wiring while the system is powered.
  • Wait for the LCD startup screen before trying to use the controls.

Quick Start

  1. Power on the Raspberry Pi and bench supply.
  2. Wait for the LCD to finish startup.
  3. Rotate the encoder to highlight Mode Select.
  4. Press the encoder to enter the menu.
  5. Rotate to the mode you want to use.
  6. Press the encoder to select it.
  7. Use Back to go up one level or Main to return to the top menu.

What This Tool Bench Can Do

  • Generate square waves
  • Generate sine waves
  • Measure resistance
  • Measure DC voltage
  • Output a DC reference voltage
  • Measure the frequency of an external sine wave

Controls and UI

The system is controlled using the LCD and a rotary encoder with a push button. All menu navigation is done with this control.

Controls

  • Rotate encoder: move the cursor through menu options
  • Press encoder: select the highlighted option
  • Back: return to the previous menu level
  • Main: return to the main screen

Main Menu

  • OFF
  • Mode Select

Mode Select Options

  • Function Generator
  • Ohmmeter
  • Voltmeter
  • DC Reference
  • Frequency Measurement
  • Back
  • Main

Navigation Tips

  • If you are unsure where you are in the menu, select Main.
  • When leaving an output mode, the output should return to its safe default off state.
  • Always check the LCD before connecting or disconnecting signals.

Function Generator

The Function Generator mode is used to create either a square wave or a sine wave. The user can select waveform type, frequency, amplitude, and whether the output is on or off.

How to Enter This Mode

  1. From Mode Select, choose Function Generator.
  2. Select Type to choose the waveform.
  3. Select Frequency to set the output frequency.
  4. Select Amplitude to set the output amplitude.
  5. Select Output and turn it On.

Square Wave

  • Centered around 0 V
  • 50% duty cycle
  • Adjustable frequency
  • Adjustable amplitude

Sine Wave

  • Positive voltage only
  • 0 V to 10 V amplitude range
  • 1 kHz to 10 kHz frequency range
  • 500 Hz frequency steps

What the Display Shows

When output is on, the LCD shows the active waveform settings so the user can confirm the selected type, frequency, and amplitude.

Important Notes

  • When you leave this mode using Back or Main, the output should default to off.
  • Make sure the output is connected to the correct labeled terminal before enabling it.
  • This output is intended to be high impedance.

Ohmmeter

The Ohmmeter measures an external resistor and displays the measured value on the LCD.

Measurement Range

  • Designed for approximately 500 Ω to 10 kΩ
  • Autoranging within the supported design range

How to Use the Ohmmeter

  1. Connect the unknown resistor to the labeled ohmmeter terminals.
  2. From Mode Select, choose Ohmmeter.
  3. Wait for the measurement to stabilize on the LCD.
  4. Read the displayed resistance value.

What the Display Means

The LCD shows the measured resistance and may also show a threshold or tolerance value depending on the current software version.

Best Practices

  • Only measure resistors within the supported range.
  • Make sure the resistor is not connected to another powered circuit.
  • Wait briefly for the reading to settle before recording the value.

Voltmeter

The Voltmeter reads a DC voltage and displays the result on the LCD. It can measure either an external voltage or the system's internal DC reference.

Measurement Range

  • Designed for approximately -5 V to +5 V DC

Source Options

  • External: reads a voltage applied at the external voltmeter input
  • Internal Reference: reads the DC Reference output through the internal path

How to Use the Voltmeter

  1. From Mode Select, choose Voltmeter.
  2. Select Source.
  3. Choose External or Internal Reference.
  4. If using external mode, connect the signal to the labeled voltmeter input.
  5. Read the displayed voltage from the LCD.

Helpful Note

If Internal Reference is selected, the interface may take you into the DC Reference path so the user can set and verify the internal source value.

DC Reference

The DC Reference mode generates a selectable DC voltage reference that can be viewed on the LCD and measured using the voltmeter.

Output Range

  • -5 V to +5 V
  • 0.625 V steps

How to Use the DC Reference

  1. From Mode Select, choose DC Reference.
  2. Select Voltage Value Input.
  3. Rotate the encoder to choose the desired voltage.
  4. Return to Output.
  5. Select On to enable the output.
  6. Use the voltmeter display to confirm the generated value.

Important Notes

  • This is a reference output, not a power supply.
  • It is intended for high-impedance use and should not be used to drive heavy loads.
  • When leaving this mode through Back or Main, the output should default to off.

Frequency Measurement

This mode measures the frequency of an external sine wave and reports the result on the LCD.

Supported Input

  • External sine wave input
  • Designed for approximately 1 kHz to 10 kHz

How to Use Frequency Measurement

  1. Connect the external sine wave source to the labeled frequency measurement input.
  2. From Mode Select, choose Frequency Measurement.
  3. Wait for the LCD reading to stabilize.
  4. Read the measured frequency value.

Best Practices

  • Use a clean sine-wave input from an external source.
  • Make sure the signal is connected to the correct input terminal.
  • If no value appears, verify that the source is active and within the supported range.

Troubleshooting and Safety

Common Problems

  • The LCD is on, but nothing changes: check the encoder wiring and restart the system.
  • No waveform output: verify that Output is set to On and the correct waveform mode is selected.
  • Voltmeter reading is incorrect: confirm the selected source and check the signal connection.
  • Ohmmeter reading is unstable: make sure the resistor is connected firmly and is not attached to a powered circuit.
  • No frequency reading: confirm that a sine wave is present and connected to the correct input.

Safety Notes

  • Do not hot-plug wiring while the system is powered.
  • Do not exceed the intended input voltage range.
  • Use the DC Reference only as a reference source, not as a load-driving supply.
  • Double-check terminal labels before turning on outputs.

FAQ

  • Why does the output turn off when I leave a menu? This is the intended safe default behavior.
  • Can I use the voltmeter on any signal? Only within the supported DC input range.
  • Can I use the ohmmeter on an in-circuit resistor? No. Remove the resistor from powered circuitry first.
