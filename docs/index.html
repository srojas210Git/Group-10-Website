Start Here

This website is the user manual for the ECSE 2920 Electronics Tool Bench built by Group 10. It is written for a first-time user and explains how to safely power the system, navigate the menu, and use each measurement and signal-generation mode.

Before You Begin

  • Only operate the workbench in dry, indoor environments.
  • Make sure the Raspberry Pi and the 24 V supply are connected correctly.
  • Make sure all external test leads are connected to the correct labeled terminals.
  • Do not unplug or modify any wiring while the system is powered.
  • Wait for the LCD startup screen before using the controls.

Quick Start

  1. Power on the Raspberry Pi and bench supply.
  2. Wait for the LCD to finish startup and display Group 10 Test Bench.
  3. Rotate the encoder to highlight Mode Select.
  4. Press the encoder to enter the menu.
  5. Rotate to the mode you want and press the encoder to select it.
  6. Use Back to go up one level, or Main to return to the top menu.
  7. Holding the encoder for 3 seconds also returns you to the start menu from anywhere.

What This Tool Bench Can Do

  • Generate square waves (100 Hz – 10 kHz, ±10 V, 50% duty cycle)
  • Generate sine waves (1 kHz – 10 kHz, 0 – 10 V)
  • Measure resistance (500 Ω – 10 kΩ, autoranging)
  • Measure DC voltage (−5 V to +5 V)
  • Output a selectable DC reference voltage (−5 V to +5 V, 0.625 V steps)
  • Measure the frequency of an external sine wave (1 kHz – 10 kHz)

The Project

Full assembled bench — front view

The completed Group 10 Electronic Tool Bench integrates a Raspberry Pi, a blue LCD display, a rotary encoder, and custom analog circuitry into a single benchtop unit.

LCD display and rotary encoder interface

System Overview

  • Microcontroller: Raspberry Pi
  • Display: Blue 16×2 LCD
  • Input: Rotary encoder with push button
  • Power rails: ±12 V (60 mA recommended, 1 A absolute max)
  • GPIO limit: 16 mA per pin
  • Outputs: High-impedance function generator and DC reference terminals
  • Inputs: Labeled terminals for voltmeter, ohmmeter, and frequency measurement

Controls and UI

All interaction is done through a rotary encoder with push-button and a blue LCD display. The cursor position is marked by > and all other options by *.

Encoder Controls

  • Rotate clockwise: move cursor down one row
  • Rotate counterclockwise: move cursor up one row
  • Press once: select the highlighted option
  • Hold 3 seconds: return to the start menu from anywhere
  • Back (menu option): return to the previous menu level
  • Main (menu option): return to the main screen

Adjusting a Value

When you select a value to edit, the encoder enters Edit Mode. Rotating slowly changes the value in small increments; rotating faster changes it by a larger amount. Press the encoder to confirm and apply the displayed value. Hold the encoder for 3 seconds to exit the edit menu and return to the previous menu.

Note: selecting DC Reference from the Voltmeter menu does not set a value — it only redirects the measurement path. This is the only exception to the press-to-set behavior.

Navigation Tips

  • If you are unsure where you are, hold the encoder for 3 seconds or select Main.
  • When leaving any output mode, the output returns to its safe default off state.
  • Always check the LCD before connecting or disconnecting signals.

Function Generator

The Function Generator produces either a zero-centered square wave or a positive-only sine wave. Connect your circuit to the high-impedance output terminal before enabling output.

How to Use

  1. Navigate to Main > Mode Select > Function Generator.
  2. Select Type and choose Square or Sine.
  3. Select Frequency > Input Frequency and use the encoder to set the desired value.
  4. Select Amplitude > Input Amplitude and dial in your peak voltage.
  5. Select Output and choose On.
  6. The LCD will display the live waveform settings while the output is active.

Square Wave

  • Centered at 0 V (symmetric positive and negative swing)
  • Fixed 50% duty cycle
  • Frequency range: 100 Hz to 10 kHz
  • Amplitude adjustable up to ±10 V

Sine Wave

  • Positive voltage only (0 V to 10 V)
  • Amplitude adjustable in 0.625 V steps
  • Frequency range: 1 kHz to 10 kHz in 500 Hz steps

Important Notes

  • This output is high impedance — do not drive heavy loads.
  • Never apply an external voltage into the function generator output terminals.
  • When leaving this mode via Back or Main, the output defaults to off.

Ohmmeter

The Ohmmeter measures an external resistor and displays the value on the LCD. It autoranges within its supported range with no manual range selection needed.

Measurement Range

  • 500 Ω to 10 kΩ (autoranging)
  • The display also shows your measurement threshold and tolerance (e.g., 1000 Ω ± Y Ω)

How to Use

  1. Connect the unknown resistor to the labeled ohmmeter input terminals.
  2. Navigate to Main > Mode Select > Ohmmeter.
  3. Wait for the reading to stabilize on the LCD.
  4. Read the displayed resistance value.

Best Practices

  • Only measure resistors within the 500 Ω – 10 kΩ range.
  • Never connect the ohmmeter to a live circuit. The component must be isolated and fully depowered before measuring.
  • Wait briefly for the reading to settle before recording the value.

Voltmeter

The Voltmeter measures DC voltage and displays the result alongside a tolerance value (e.g., 2.5 V ± 0.3125 V). It can read an external circuit or verify the internal DC Reference output.

Measurement Range

  • −5 V to +5 V DC

Source Options

  • External: measures a voltage at the external voltmeter input probes
  • Internal Reference: reroutes the voltmeter to read the DC Reference output through the internal path

How to Use

  1. Navigate to Main > Mode Select > Voltmeter.
  2. Select Source and choose External or Internal Reference.
  3. If using External, connect your signal to the labeled voltmeter input.
  4. Read the live voltage and tolerance on the LCD.

Note

If Internal Reference is selected, the interface may navigate into the DC Reference path so you can set and verify the internal source value. Pressing a value in this path does not set it — this is expected behavior.

DC Reference

The DC Reference generates a precise, selectable DC voltage that can be verified live with the internal voltmeter. This is a high-impedance reference source and is not designed to supply current to power heavy loads.

Output Range

  • −5 V to +5 V, adjustable in 0.625 V steps

How to Use

  1. Navigate to Main > Mode Select > DC Reference.
  2. Select Voltage Value Input and rotate the encoder to your desired level.
  3. Press the encoder to confirm the value.
  4. Select Output and choose On.
  5. The UI will use the internal voltmeter to display the live output voltage for verification.

Important Notes

  • This is a reference output, not a power supply. Do not drive heavy loads.
  • When leaving this mode via Back or Main, the output defaults to off.

Frequency Measurement

This mode measures the frequency of an incoming external sine wave and displays the live reading on the LCD.

Supported Input

  • External sine wave only
  • Calibrated for 1 kHz to 10 kHz

How to Use

  1. Connect the external sine wave source to the labeled frequency measurement input terminal.
  2. Navigate to Main > Mode Select > Frequency Measurement.
  3. Wait for the LCD reading to stabilize.
  4. Read the measured frequency value.

Best Practices

  • Use a clean sine wave from a known source.
  • Confirm the signal is at the correct input terminal.
  • If no value appears, verify the source is active and within the 1 kHz – 10 kHz range.

Safety Instructions

General Operating Environment

  • Operate only in dry, indoor environments. High humidity, extreme heat, or extreme cold can cause malfunctions and affect readings.
  • Do not unplug, modify, or rewire the workbench in any way, especially while powered.

Power Supply & Thermal Limits

  • Maximum recommended current on the ±12 V rails: 60 mA. Absolute maximum: 1 A.
  • Maximum current per GPIO pin: 16 mA — do not exceed this.
  • The power supply can become very hot under load. Do not touch internal components while powered.
  • Wait at least 5 minutes after unplugging before touching any internal components.

Signal Input Limits

  • Do not exceed the rated input voltage range on any terminal.
  • Never apply an external voltage into the function generator output terminals.
  • Ohmmeter: always isolate and depower the component before measuring. Never connect to a live circuit.

Emergency Procedure

  • If the bench smokes, smells unusual, sparks, or behaves erratically — immediately disconnect power and step back.
  • Do not touch the circuit until at least 10 minutes have passed and all signs of active hazards are gone.

Troubleshooting & FAQ

Common Problems

  • LCD is on but nothing responds: check the encoder wiring and restart the system.
  • No waveform output: verify Output is set to On and the correct waveform type is selected.
  • Voltmeter reading looks wrong: confirm the selected source (External vs. Internal Reference) and check the signal connection.
  • Ohmmeter reading is unstable: ensure the resistor is firmly connected and completely isolated from any powered circuit.
  • No frequency reading: confirm a sine wave is present at the correct input terminal within the 1 kHz – 10 kHz range.
  • DC Reference output seems off: use the Internal Reference voltmeter path to verify the actual output voltage.

FAQ

  • Why does the output turn off when I leave a menu? This is intentional safe default behavior to prevent unintended signals.
  • Can I use the voltmeter on any signal? Only DC signals within −5 V to +5 V. Do not exceed the input range.
  • Can I measure an in-circuit resistor? No. Remove it from any powered circuit and isolate it completely before measuring.
  • What does holding the encoder do? Holding for 3 seconds returns you to the start menu from anywhere in the menu.
  • Why won't a value set when I select DC Reference from the Voltmeter menu? That path only redirects the measurement source — it does not set a voltage value. This is expected behavior.
  • Can the DC Reference power a circuit? No. It is a high-impedance reference only and is not designed to drive loads.
